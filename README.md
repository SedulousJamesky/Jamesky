# Blog

A modern blog built with Next.js and MDX.

## Setup

```bash
npm install
npm run dev
```

## Writing Posts

Create `.mdx` files in `src/content/posts/` with frontmatter:

```yaml
---
title: My First Post
date: "2024-01-15"
description: A short description
---
```

## Build

```bash
npm run build
```
